ПУБЛИЧНЫЕ:
GET /program - Список всех событий
GET /program/{event} - Детали события (NEW!)
GET /news - Список всех новостей
GET /news/{news} - Полная статья (NEW!)
GET /partners - Страница партнеров (УЛУЧШЕНА!)

API:
POST /api/events - Создать событие (для admin)
GET /api/events - Получить все события
GET /api/events/{id} - Детали события (JSON)
PUT /api/events/{id} - Обновить событие
DELETE /api/events/{id} - Удалить событие

Аналогично для /api/news и /api/partners

1. Запустите проект:
   php artisan serve

2. Откройте в браузере:

    СОБЫТИЯ:

    - http://localhost:8000/program (список)
    - http://localhost:8000/program/1 (детали события #1)

    НОВОСТИ:

    - http://localhost:8000/news (список)
    - http://localhost:8000/news/1 (статья #1)

    ПАРТНЕРЫ:

    - http://localhost:8000/partners (все партнеры)

3. Попробуйте функции:
    - Нажимайте кнопки социального шеринга
    - Переходите по ссылкам на другие события/новости
    - Просмотрите информацию о партнерах
